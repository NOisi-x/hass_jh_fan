# JH 风扇 — Home Assistant 集成

由生成式AI基于小程序逆向分析的 JH语音 风扇蓝牙集成，适用于可由 JH语音 小程序控制的蓝牙BLE协议风扇。
已验证的设备：奥克斯空气循环扇语音版（FX-L36AR-V）

## 功能

| 实体 | 类型 | DP 键 | 说明 |
|------|------|-------|------|
| 风扇 | `fan` | switch / level_1 | 开关、调速、左右摇头 |
| 氛围灯 | `switch` | light_1 | 氛围灯开关 |
| 驱蚊模式 | `switch` | mosquitoControl | 驱蚊功能 |
| 语音播报 | `switch` | voiceaAnnounce | 语音播报开关 |
| 上下摇头 | `switch` | angleAutoUDOnOff | 上下摇头开关 |
| 定时关机 | `number` | timingPowerOff1 | 0–12 小时倒计时 |

**可配置最大档位** — 添加设备时根据设备实际情况选择匹配的最大风速档位范围使百分比调速时正确映射：3 / 6 / 8 / 12 / 32 / 36（默认 8）。

## 环境要求

需Home assistant有蓝牙硬件支持，推荐使用 ESP32 蓝牙代理

## 安装

### HACS

1. 打开 HACS → **集成**
2. 点击 **⋮** → **自定义存储库**
3. 输入仓库地址，类型选 **集成**
4. 搜索 **JH Fan** → **下载**
5. 重启 Home Assistant

## 添加设备

**设置 → 添加集成 → JH 风扇** → 选择方式：

1. **扫描设备** — BLE 扫描，从列表中选择
2. **蓝牙自动发现** — HA 被动检测，弹出确认
3. **手动输入** — 直接输入 MAC 地址

选择设备后，选择与实际型号匹配的最大档位。

## 多设备

每个风扇独立添加为单独的配置条目，所有实体按 MAC 地址隔离，互不冲突。

## 状态同步

| 阶段 | 时机 | 机制 |
|------|------|------|
| 初始查询 | 连接时 | query-all |
| 乐观更新 | 0ms | 本地缓存 → 即时 UI |
| 延迟验证 | ~0.5s | query-all → 权威数据 |
| 定期轮询 | 30s | 全量状态查询 |
| 心跳保活 | 20s | 防止 BLE 超时 |


## 常见问题

| 现象 | 解决方法 |
|------|----------|
| 扫描不到设备 | 确保设备已开机，在蓝牙范围内 |
| 连接断开 | 缩短距离，检查 ESP32 信号 |
| 档位不对 | 重新添加设备，选择正确最大档位 |
| 状态不同步 | 集成每 30 秒自动同步 |
